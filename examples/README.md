# Examples

Curated examples are organized by language:

- `examples/python/`
- `examples/typescript/`

## Python

- `examples/python/basic_trace.py` - minimal trace + nested spans via Python SDK
- `examples/python/local_trace.py` - generate many synthetic local traces
- `examples/python/eval_example.py` - dataset + queue review loop
- `examples/python/claude_code_example.py` - call MCP tools over HTTP JSON-RPC

## TypeScript

- `examples/typescript/quick_trace.ts` - minimal trace + span flow via HTTP
- `examples/typescript/mcp_client.ts` - call MCP `tools/list` and `search_traces`
- `examples/typescript/vercel_ai_sdk.ts` - Vercel AI SDK instrumentation
